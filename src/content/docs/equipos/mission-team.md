El Mission Team ejecuta sus responsabilidades mediante un ciclo iterativo que implementa continuamente el modelo de 10-Pasos para construir valor de manera incremental.

## Funcionamiento Autónomo

El equipo opera con autonomía en sus decisiones, considerando siempre los habilitadores y principios establecidos para cada ciclo.

**Habilitadores:**
- Timeboxes
- Outcomes

## Principios Operacionales

- **Visión clara**: Definir qué se construirá en el ciclo y cuál es el outcome esperado.
- **Sentido de Urgencia**: Entregar valor cada 15 días asociado a los outcomes; reevaluar outcomes cada 3 meses para adaptar el siguiente ciclo.
- **Disrupciones constantes**: Cuestionar el status quo para que el valor emerja desde perspectivas no convencionales.

## Estructura Metodológica

El equipo se basa en el ciclo de [10 Pasos](/evolucion/10-pasos/) para ejecutar incrementalmente sus construcciones, manteniendo enfoque en outcomes y timeboxes como elementos clave del proceso.
