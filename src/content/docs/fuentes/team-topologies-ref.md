Recursos educativos y materiales de referencia sobre Team Topologies para profundizar en el framework.

## Guía de Facilitación de Club de Lectura

Documento colaborativo para la facilitación del libro *Team Topologies*.

- **Autores:** Magda Björkman y Viktor Cessan
- **Formato:** Google Docs compartido
- Incluye dinámicas y preguntas para guiar discusiones grupales capítulo por capítulo

## Análisis y Revisión Rápida

- Reseña detallada del libro *Team Topologies* con análisis de sus conceptos centrales
- "Quick Reference Card" (QRC) visual con las topologías y modos de interacción
- **Fuente:** Blog de Henny Portman
- Incluye gráficos ilustrativos sobre las cuatro topologías de equipos

## Conceptos Clave para Referencia Rápida

| Topología | Propósito |
|---|---|
| Stream-Aligned | Entrega independiente orientada al flujo de valor |
| Platform | Infraestructura y servicios internos de soporte |
| Enabling | Facilita adopción de nuevas prácticas en otros equipos |
| Complicated-Subsystem | Gestiona subsistemas de alta complejidad técnica |

| Modo de Interacción | Cuándo usar |
|---|---|
| Colaboración | Cuando ambos equipos necesitan trabajar juntos para descubrir soluciones |
| X-as-a-Service | Cuando un equipo ofrece servicios claros y documentados |
| Facilitación | Cuando un equipo ayuda a otro a crecer en capacidad |
